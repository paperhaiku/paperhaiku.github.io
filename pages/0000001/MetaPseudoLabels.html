Haiku
Teacher is pre-trained on labelled data,
Teacher makes pseudo labels
From unlabelled data.

Student learns from pseudo labels
Student predicts labelled data
Student loss = CE(Teach Pred, Stud Pred)

Jointly train
Teacher and student,
Teacher loss = CE(Teach Pred, GT) + Student loss

Every iteration, teacher learns
from student's mistakes,
as well as labelled data.
Abstract
We present Meta Pseudo Labels, a semi-supervised learning method that achieves a new state-of-the-art top-1 accuracy of 90.2% on ImageNet, which is 1.6% better than the existing state-of-the-art. Like Pseudo Labels, Meta Pseudo Labels has a teacher network to generate pseudo labels on unlabeled data to teach a student network. However,unlike Pseudo Labels where the teacher is fixed, the teacher in Meta Pseudo Labels is constantly adapted by the feedback of the studentâ€™s performance on the labeled dataset. As a result, the teacher generates better pseudo labels to teach the student
